# Como ler o Lawfare Timeline — guia para o leitor brasileiro

Mais de **1.600 entradas** documentadas entre 1855 e 2026. Ótimo para quem investiga a fundo — complicado para quem quer entender, em quinze minutos, o que está acontecendo no país. Este artigo é a porta de entrada.

Ele faz quatro coisas: explica **como o projeto é organizado**; aponta os **5 artigos essenciais**; sugere **3 leituras por perfil** (cidadão, investigador, juiz, delegado, diplomata); e lista os **3 melhores por categoria**, com uma frase sobre cada um.

## O problema que este projeto tenta resolver

A maior parte do que um brasileiro sabe sobre justiça, corrupção e poder chega filtrada por dois funis: o **noticiário do dia** (que esquece rápido) e a **cobertura editorial** de veículos com contratos publicitários, dependência de acesso a fontes oficiais, ou receio de desagradar quem os processa. Não é teoria da conspiração — é rotina econômica de redação.

O efeito prático tem nome aqui: **P04 (weaponização da mídia e da narrativa)**. Um caso vira manchete por dias e some sem desfecho. Um número gigantesco (R$ 86 bilhões, R$ 51 bilhões) aparece uma vez e nunca mais é atualizado.

A variação mais sutil é **P04b (both-sidesism funcional)**: em vez de esconder o fato, a cobertura o **dilui** — trata padrão documentado como "só mais uma opinião", equipara dado a narrativa, ou desloca o debate dos fatos para a motivação de quem denuncia. Dois exemplos concretos:

- Designação de PCC/CV como terrorista pelos EUA (2026) noticiada como **"ameaça à soberania"**, não pelos fatos (indiciamentos, sanções, células em 12 estados americanos). [Artigo P04b →](https://lawfare-timeline.vercel.app/posts/imprensa-brasileira-enquadra-designacao-terrorista-pcc-cv-como-questao-de-soberania-p04b/)
- Relatório de observatório alinhado à direita repete o mesmo mecanismo — trocando só o alvo. [T-227 P04 pela Direita →](https://lawfare-timeline.vercel.app/posts/2026-07-20-p04-pela-direita-narco-soberania-eleitoral-e-diplomacia-das-sombras-como-espelhos-estrutur/)

> Este site não pede confiança em opinião nenhuma — nem na dele. Cada entrada tem fonte, data e selo: `ev-confirmed`, `ev-contested`, `ev-alleged`, `ev-inference`. Quando ler algo chocante, o primeiro instinto certo é clicar na fonte.

## Três camadas de organização

**Camada 1 — Linha do tempo.** Eventos pontuais: decisão, prisão, denúncia. Uma pasta por assunto (`bancos`, `stf`, `tse`, `justica`…). Cada card tem data, fonte e, quando aplicável, ID sequencial (`id_corpus: 1635`).

**Camada 2 — Estudos.** Análises que cruzam várias entradas e nomeiam mecanismos recorrentes — a resposta para "por que isso se repete?".

**Camada 3 — Padrões sistêmicos (P01–P11).** Um padrão só existe se aparecer em **três ou mais casos independentes**, com o mesmo mecanismo:

| Código | Em uma frase |
|--------|--------------|
| P01 | O processo desmorona por defeito formal, não porque o fato não aconteceu |
| P02 | Quem denuncia vira réu; o investigado vira "vítima" |
| P03 | Um ministro do STF, sozinho, destrava ou trava tudo |
| P04 | A cobertura decide o que "aconteceu", não o fato |
| P04b | A cobertura dilui o fato como "só um lado da história" |
| P05 | Dinheiro público vira veículo do esquema que deveria fiscalizar |
| P06 | Nada é absolvido nem condenado — o caso envelhece até prescrever |
| P07 | A mesma rede de proteção atravessa gerações e governos |
| P08 | Crime organizado usa fintech/cripto para lavar em escala |
| P09 | Captura da legitimidade cultural — quem denuncia perde prestígio |
| P10 | Política e crime compartilham infraestrutura de serviço |
| P11 | Juros, dívida e austeridade se realimentam sem coordenação explícita |

## Regra prática de leitura

1. Comece pelos **5 artigos abaixo**.
2. Vá para **seu perfil** (cidadão, investigador, juiz, delegado, diplomata).
3. Use a lista **por categoria** como consulta — ninguém lê 1.600 entradas de uma vez.
4. Quando uma manchete parecer "óbvia demais", é hora de checar a fonte.

## A regra final

Nomear manipulação de mídia não dá licença para inventar um novo viés. Descobrir que "a imprensa engana" e passar a desconfiar só do que contraria o que você já acha é o P04b ao contrário. A régua é sempre a mesma: **fonte, data, e o que pode ser checado**. Quando um artigo aqui não tiver isso, desconfie dele também.
